# 🎬 StreamBuzz — Movie Website Landing Page

StreamBuzz is a **responsive movie website landing page** built using **HTML + CSS + Bootstrap**. It showcases a modern UI with a **navbar**, a **hero section**, **movie posters carousel**, **genres carousel**, an **About Us** section, and a simple **Contact + Footer** layout. The goal of this project is to create a clean, scroll-friendly landing page that looks good on both desktop and mobile.
